# 🎨 Guide d'utilisation des couleurs dans l'éditeur

## Comment colorer du texte

### ✅ Méthode correcte pour colorer des mots individuels :

1. **Sélectionnez le texte** que vous voulez colorer :
   - Sur mobile : Appuyez longuement sur un mot, puis faites glisser les poignées pour sélectionner le texte
   - Sur ordinateur : Cliquez et glissez sur le texte

2. **Cliquez sur l'icône palette** 🎨 dans la barre d'outils
   - L'icône affiche la dernière couleur utilisée

3. **Choisissez une couleur** dans le sélecteur

4. **La couleur s'applique uniquement au texte sélectionné** ✨

### 📝 Exemple d'utilisation :

Pour écrire : "Le ciel est **<span style="color: #4ECDC4">bleu</span>** et le soleil est **<span style="color: #FFEAA7">jaune</span>**"

1. Tapez : "Le ciel est bleu et le soleil est jaune"
2. Sélectionnez le mot "bleu"
3. Cliquez sur la palette et choisissez le bleu turquoise
4. Sélectionnez le mot "jaune"
5. Cliquez sur la palette et choisissez le jaune
6. Terminé ! 🎉

### ⚠️ Important à savoir :

**Comportement de l'éditeur riche :**

- **Si vous sélectionnez du texte** → La couleur s'applique uniquement au texte sélectionné ✅
- **Si aucun texte n'est sélectionné** → La couleur s'appliquera au prochain texte que vous taperez

### 💡 Astuce :

Pour colorer plusieurs mots de couleurs différentes :
1. Écrivez d'abord tout votre texte
2. Ensuite, sélectionnez et colorez chaque partie individuellement

## 🔧 Dépannage

### Problème : Toute la ligne change de couleur
**Solution :** Assurez-vous de bien sélectionner uniquement le texte que vous voulez colorer AVANT de cliquer sur la palette de couleurs.

### Problème : La couleur ne s'applique pas
**Solution :** Vérifiez que vous avez bien sélectionné du texte dans l'éditeur (vous devriez voir les poignées de sélection).

### Problème : Je veux enlever la couleur
**Solution :** Sélectionnez le texte coloré et choisissez la couleur blanche dans le sélecteur.
